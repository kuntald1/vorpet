Feature 04 — Auto Inventory Deduction

Bill generated. Stock auto-deducted. Zero effort.

Link a recipe to each menu item. The moment a bill is generated, CurryCloud looks up the recipe, calculates ingredient quantities, and deducts from the correct branch's stock node — automatically. Even works offline and syncs when back online.

✓
Recipe-based deduction — per ingredient, per qty
✓
Branch-to-node mapping — each branch deducts its own stock
✓
Works offline — syncs deduction when internet returns
✓
Consumption record with order reference logged

Offline-First Technology

Never lose a sale because of bad internet

Most POS systems crash when the internet goes down. CurryCloud was built offline-first — your restaurant keeps running no matter what.

📦
Cached menu & tables
All menu items, categories, and tables cached locally.
🧾
Full billing offline
SGST, CGST, surcharge, discount, round-off — all offline.

Smart auto-sync
Orders, bills, and items sync automatically when back online.
📊
Sync progress counter
Real-time progress bar showing how many orders remain.
How sync works
1
Internet drops — Offline mode activates instantly. Red banner appears.
2
Orders continue — Create orders, add items, generate bills locally.
3
Print receipts — Offline bills print with all amounts, GST, payment details.
4
Internet returns — Sync starts automatically with progress bar.
5
All synced ✓ — Orders, bills, and items appear in reports. Nothing lost.
